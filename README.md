## Опис
Цей репозиторій містить матеріали до лабораторної роботи №6 з дисципліни Python.

**Тема роботи:**  
«Обробка текстових даних, кодувань та серіалізація об'єктів у Python».

У ході виконання лабораторної роботи було створено 3 програми:
1. Програма для сортування тексту з урахуванням українського та латинського алфавітів.
2. Програма для декодування інтернет-посилань, які містять кириличні символи.
3. Програма для запису та читання JSON-файлу з українськими символами у кодуванні UTF-8.

## Структура проєкту
```text
LR6_Python_Txt/
├── input_text.txt
├── program1_sort_text.py
├── program2_decode_url.py
├── program3_json_ua.py
├── students.json
└── requirements.txt
```

## Файли
- **input_text.txt** — текстовий файл, який використовується у програмі 1.
- **program1_sort_text.py** — програма сортування слів за заданими правилами.
- **program2_decode_url.py** — програма для декодування URL-адрес.
- **program3_json_ua.py** — програма для запису та читання JSON-файлу.
- **students.json** — створений JSON-файл з даними студентів.
- **requirements.txt** — список встановлених пакетів у віртуальному середовищі.

## Запуск
Перед запуском програм потрібно активувати віртуальне середовище:

```powershell
& ..\Scripts\Activate.ps1
```

Після цього програми запускаються такими командами:

```powershell
python .\program1_sort_text.py
python .\program2_decode_url.py
python .\program3_json_ua.py
```
